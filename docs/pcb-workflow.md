# PCB作成〜基板発注までの手順

回路図(完成済み)からJLCPCBへの発注までの作業フロー。EasyEDA Pro(英語UI)のメニュー操作ベース。

## 0. 前提(完了済み)

- 回路図完成、ネットリスト検証済み
- 部品はLCSCコード紐付け済み(Device Standardizationパネルで在庫確認可能)
- 部品リスト: [bom.md](bom.md)

## 1. 回路図側の最終チェック

1. 回路図エディタで **Design → Check DRC**(回路図のルールチェック)を実行し、未接続ピン・重複デジグネータ等の警告を潰す
   - 意図的な未接続(ADS1115のALERT/RDY、XIAOの未使用ピン等)は無視してよい
2. **Advanced → Device Standardization** で在庫を最終確認(「LCSC Stock / SMT Stock」の右側=JLCPCB実装用在庫を見る)

## 2. 回路図→PCBへ転送

1. 回路図エディタで **Design → Update PCB**(プロジェクト内のPCB1へ増分反映)
2. PCB画面に切り替わり、全部品(フットプリント)が基板外に固まって配置される
3. 以降、回路図を修正した場合はそのつど **Design → Update PCB** で再同期する(PCB側の配置・配線は保持される)

## 3. 基板外形(アウトライン)を引く

1. PCB画面で **Place → Board Outline → Rectangle** を選び、**大体のサイズで矩形を引く**(あとから正確にする)
2. アウトラインを選択 → 右側プロパティパネルで座標・幅・高さを数値指定して確定
   - 目安: 3.5mmジャック12個を一辺に並べるため、横幅は最低でも 12 × 12mm ≒ 150mm 程度は必要(2列に分けるなら半分)
   - 角は角丸(R2〜3mm)にしておくとケース製作が楽
3. **Place → Hole** で四隅に取付穴(M3ならΦ3.2、少し余裕を持たせるなら3.4mm)を配置

### 3.1 穴あけの具体的な手順

- **ネジ止め用の非導通穴(NPTH)**: `Place → Hole` でクリック配置 → 右側プロパティパネルで直径を数値入力。基板端からのクリアランスはJLCPCB基準で≥1mm確保(穴の外周にも銅箔・シルクを置かない)
- **座標を揃えたい場合**: 1個配置したあと、プロパティパネルでX/Y座標を直接数値入力するのが早い。4隅を対称にしたいなら基板外形の原点(左下など)からのオフセットで計算して入力する
- **導通させたい穴(基板固定用スタンドオフをGNDに落としたい等)の場合**: 生のHoleではなく、`Place → Part` でJLC標準ライブラリの `MountingHole` 系フットプリント(検索窓に"MountingHole"や"M3")を置くと、穴径・銅リング・クリアランスがルール込みで用意されているので楽。今回はケース固定のみが目的なら通常のNPTHで十分
- 4隅配置後、`Edit → Align → Distribute`(下記5.1参照)で位置を微調整すると綺麗に揃う

## 4. デザインルール設定

**Design → Design Rule** でJLCPCB 2層標準に合わせる:

| 項目 | 値 |
|---|---|
| 最小トレース幅/クリアランス | 6mil(0.152mm)以上 ※余裕があれば10mil |
| ビア | 穴Φ0.3mm / ランドΦ0.6mm以上 |
| 信号線幅 | 10mil程度 |
| 電源(3.3V)・GND | 20mil以上 |
| VBIAS | 15mil程度(12chに分配するため太めに) |

ネットクラスを分けて(VCC 3V3 / GND / VBIAS / 信号)ルールを割り当てると管理しやすい。

## 5. 部品配置(手動)

オートルート前に配置で8割決まる。このボードの配置指針:

1. **CN1〜CN12(ジャック)**: 基板の一辺(または対向2辺に6個ずつ)に等間隔で並べ、挿抜方向を基板外向きに。CTケーブルが12本束になるので隣接ピッチに余裕を持たせる
2. **各chの入力回路(22Ω・トリマー・100nF)**: 対応するジャックの直近にchごとにまとめる。**トリマーは調整ドライバーが上から入る向き**に統一
3. **ADS1115(U3/U4/U5)**: 担当4chグループの中央付近に。**デカップリング(C13〜C15)は各ICのVDDピン直近**(数mm以内)
4. **U2(LMV358)+分圧(R28/R29)+C16/C17**: 基板中央寄り(VBIASを12chへ分配する起点なので偏らせない)
5. **U1(XIAO)**: USB-Cコネクタが基板端(ケース開口部)を向くように配置。**アンテナ部は基板端から外にはみ出すか、下層の銅箔・配線を禁止(キープアウト)**
6. **LED1**: ケースの窓/導光位置に合わせる。**TP1/TP2**: ワイヤーをハンダ付けしやすい端に
7. 回路図上の配置グループをPCBに転写したい場合は **Layout → Layout Transfer** が使える(回路図で選択した部品群の相対配置をPCB側に再現)

### 5.1 配置を効率化するツール(グループ化・自動コピー・吸着)

このボードはCT入力回路(ジャック+22Ω+トリマー+100nF)を12ch分繰り返すので、素直にやると同じ配置作業を12回やることになる。以下のツールで短縮できる:

- **グループ化**: 1ch分(ジャック+R+トリマー+C)の配置・向きを決めたら、それらをドラッグ選択(またはShiftクリック)して右クリック → **Group**(またはCtrl+G)でひとまとまりにする。以降まとめて移動・回転でき、うっかり1個だけズレる事故を防げる
- **配置の自動コピー(アレイ)**: グループ化した1ch分を選択 → 右クリック → **Create Array**(または**Edit → Array**)で、コピー数・X/Yピッチを指定すると等間隔で複製できる。12ch分を1回の操作で並べられる。ただし複製されるのは「フットプリントの配置」だけで、ネット(どのジャックがどのADS1115のどのピンに繋がるか)は回路図側の接続情報のまま変わらないので、配線(ratsnest線)がどの実体に繋がっているかは複製後に必ず目視確認すること
- **整列・等間隔配置**: 12個のジャックをきれいに並べたい場合は、まとめて選択して **Edit → Align → Left/Top等**(端を揃える)+ **Distribute**(等間隔にする)を使うと、手動でピッタリ揃えるより速くて正確
- **吸着(スナップ)**: 画面下部のステータスバーにグリッド間隔の表示があり、クリックで変更できる(例: 0.5mm/0.1mm/1milなど)。部品の粗配置は大きめのグリッド(0.5〜1mm)、ジャックなどコネクタのピッチ合わせは部品のピン間隔に合わせた細かいグリッド、配線時はさらに細かくする、というように作業内容に応じてグリッドを切り替えると位置ズレを防げる。特定の点(既存部品のピン・端点)にピタッと吸着させたい時は、部品を掴んだままカーソルを近づけるとハイライトされるオブジェクトスナップが効くのでそれに任せてよい

### 5.2 自動配置(オートプレースメント)について

EasyEDA Proには **Layout → Auto Placement** 相当の自動配置機能がある。ただし正直なところ、このボードにはあまり向かない。

- 自動配置は「部品同士が重ならないように散らす」「配線長を短くする」程度の最適化しかせず、**このボード特有の意味的な要件(ジャックは基板端向き、トリマーは調整ドライバーが入る向き、アンテナ直下は避ける、デカップリングは対象ICの直近)までは考慮しない**
- 5.1のGroup機能で「ch単位のまとまり」を作っていても、自動配置がそのまとまりを維持したまま動かしてくれるとは限らない(ツールのバージョン・設定に依存し、確実な保証はできていない)
- 実用上は、**Update PCB直後に部品が基板外で重なって出てきた状態を大まかに散らす一次処理**として自動配置を使い、そのあと5節の指針に沿って手動(グループ化+アレイ+整列)で配置し直すのが安全。自動配置だけで完成させようとしない方がいい

## 6. 配線

1. **まずオートルートで引く: Route → Auto Route**(クラウドオートルータ)。設定はデフォルト+上記ルールでOK
2. オートルートは信号線用と割り切り、**GNDは配線せずベタで処理**する前提で:
   - **Place → Copper Pour** で両面にGNDベタを作成(底面優先)。ベタ再構築は右クリック → Rebuild(配線やパターンを変更するたびに手動で再構築が必要。自動追従はしない)
   - 3.3Vは太めの手動配線 or 部分ベタ
   - THT部品(ジャック・トリマー)のパッドは手ハンダ前提なので、Copper Pourのプロパティで**サーマルリリーフ(スポーク接続)が有効になっているか確認**。ベタに直結(ソリッド接続)されているとハンダごての熱がベタに逃げて半田付けしづらくなる
3. 手直しポイント:
   - **CT入力(アナログ)配線はI2C・電源から離す**。並走させない
   - VBIASの分配は幹→枝のツリー状に(ループを作らない)
   - XIAOアンテナ直下のベタ・配線を除去
   - デカップリングコンデンサ経由で電源がICに入る形に(コンデンサを迂回しない)
   - **インピーダンス整合は今回気にしなくてよい**。差動高速信号(USBの内部配線はXIAOモジュール内で完結)やRFはこの基板では扱わず、I2C(最大100kHz程度)とCT電流信号(50/60Hz帯)はどちらも低速なので、2層の一般的な配線幅・クリアランスの範囲で問題ない。特性インピーダンス計算やその通りのトレース幅設計は不要

## 7. チェック

1. **Design → Check DRC** でエラー0にする(クリアランス違反・未配線ネットの確認)
2. 未配線が残っていないか Design Managerパネルのネット一覧でも確認
3. **View → 3D** で部品干渉・向きを目視確認(特にジャックの向き、トリマーの調整口、XIAOのUSB-C)

## 8. シルク調整

### 8.1 操作方法

- **デジグネータ(部品ごとのR1やU3等の文字)の移動**: 部品本体ではなく文字ラベル自体をクリックしてドラッグすると独立して動かせる(部品を動かすと文字も一緒について来るが、文字だけの再配置も可能)。重なりが多い場所は1つずつ手動でずらすしかない(EasyEDA Proに一括自動整列コマンドがあるかは未確認。無ければ地道に手動)
- **任意の文字を追加(CH1〜12ラベルや基板名など)**: `Place → String`(テキスト配置)でクリックした位置にテキストを置ける。プロパティパネルで文字列内容・レイヤー(Top/Bottom Silkscreen)・高さ・線幅を指定する
- **推奨サイズ**: JLCPCBの製造下限を考えると、文字高さは1mm以上、線幅0.15mm(6mil)以上を目安にする。これより小さいと製造時に潰れて読めなくなることがある
- **注意点**: シルクをパッドやビアの上に重ねない(製造時にそこだけシルクが乗らず消える)。部品の実装後に隠れる位置(ジャックの真下など)も避ける

### 8.2 やること

- デジグネータの位置・向きを整理(部品に重ならないように)
- **各ジャック横にch番号(CH1〜CH12)を印字**(運用時のブレーカー対応表と紐付けるため必須)
- 基板名・バージョン・日付を空きエリアに

## 9. 製造データ出力と発注

1. **Fabrication → PCB Fabrication File(Gerber)** で出力(またはワンクリック発注: **Fabrication → Order PCB**)
2. JLCPCB発注設定の目安:
   - 層数: 2層 / 板厚: 1.6mm / 銅箔: 1oz / 表面処理: HASL(鉛フリー)で十分
   - 数量: 5枚〜(最小ロット)
3. SMT実装サービスを使う場合:
   - **Fabrication → BOM** と **Fabrication → Pick and Place** を出力して依頼
   - Basic部品(抵抗・コンデンサ・LED)は追加費用なし。ADS1115・LMV358・PJ-320DはExtended(部品ごとに段取り費)
   - **手実装が必要な部品**: XIAO ESP32C6(SMT在庫なし)、トリマー3266W(スルーホール)、TP1/TP2への配線
4. 全部手ハンダの場合はGerberのみ発注し、部品はLCSC/秋月等で調達([bom.md](bom.md)参照)

## 10. 実装後の立ち上げ

1. 目視+テスターで電源ライン(3.3V-GND間ショートがないこと)を確認してからUSB-C接続
2. VBIASが約1.65Vであることを確認(TP2基準でU2出力を測定)
3. ESPHomeファーム書き込み([firmware/esphome/esphome-sample.yaml](../firmware/esphome/esphome-sample.yaml)を12ch用に拡張)
4. I2Cスキャンで0x48/0x49/0x4Aの3デバイスが見えることを確認
5. 校正: トリマーでレンジ調整 → クランプメーターと比較して`calibrate_linear`調整([calibration.md](calibration.md))
