# 電力センサーモジュール 基板設計仕様書(ドラフト)

## 1. 目的
分電盤の各ブレーカーの電流(または電力)をCTクランプで非接触測定し、
ESPHome経由でHome Assistantに取り込む。

## 2. 確定済みの構成

| 項目 | 内容 | 備考/根拠 |
|---|---|---|
| マイコン | Seeed Studio XIAO ESP32C6 | Wi-Fi 6, ESPHome対応 |
| I2Cピン | SDA=D4(GPIO4), SCL=D5(GPIO5) | [Seeed公式Wiki](https://wiki.seeedstudio.com/xiao_pin_multiplexing_esp32c6/) |
| ADコンバータ | ADS1115(16bit, 4ch, I2Cアドレス0x48〜) | 1個で最大4chまで独立測定可能 |
| センサー | CTクランプ SCT-013-000(YHDC) | 巻数比2000:1, 最大100A→50mA |
| ファームウェア | ESPHome | `ads1115:` + `sensor: platform: ct_clamp` |
| 測定方式 | シングルエンドモード(GND基準)+バイアス電圧+負荷抵抗 | 差動モードは2ch消費するため非採用(参考記事に準拠) |
| 参考実装 | [maky-baさんのブログ記事](https://maky-ba.hatenablog.com/entry/2025/10/08/055950) | 回路構成・キャリブレーション手法の元ネタ |

## 3. 回路構成の要点(参考記事ベース)

- 3.3VとGNDの間に10kΩ×2(R1, R2)でバイアス電圧を生成し、CT信号の負側を持ち上げる
- CTクランプの負荷抵抗(R3)は100Ω前後(E24系列)で計算(最大電流により再計算が必要)
- ADS1115はシングルエンドモード(A0_GND等)、gain設定は測定電流レンジに応じて調整
- `calibrate_linear`フィルタで実測校正(クランプメーターとの比較測定が必要)

## 4. 確定した仕様(2026-07-01決定)

### 4.1 測定チャンネル数
- CTクランプ3本(分電盤の3ブレーカー分)
- ADS1115 1個(4ch)で対応、1chは予備

### 4.2 測定する物理量
- 電流(A)のみ実測し、想定電圧(例:100V)×電流で電力(W)を近似する
- 電圧センサー(ZMPT101B等)は追加しない

### 4.3 バイアス回路の設計
- 3.3V/2分圧のバイアス電圧をオペアンプ(ボルテージフォロワ)でバッファリングし、3ch共有の1系統として供給
- オペアンプは単電源対応の汎用品(MCP6001/LM358系)を採用

### 4.4 電源
- USB-C給電のみ(ACアダプタ経由の常設電源は不採用)

### 4.5 CTクランプ接続コネクタ
- JSTコネクタを採用

### 4.6 部品の実装方式
- ADS1115・XIAO ESP32C6ともに基板へ直接ハンダ付け(ICチップ/モジュール直付け、ソケット・ピンヘッダなし)

### 4.7 筐体・設置形態
- 独立ケース(3Dプリント等)を想定。DINレール取付は不採用
- 基板サイズの制約は特になし(部品配置に余裕を持たせる)

### 4.8 キャリブレーション運用
- 個体ごとに実測校正(クランプメーターとの比較測定)を行い、`calibrate_linear`で調整する運用

### 4.10 基板仕様
- 2層基板、JLCPCB発注
